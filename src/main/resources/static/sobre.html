Sobre o Quiz Mini Games

Jogos gratuitos, direto no navegador — sem cadastro, sem download.

🎮 O projeto

O Quiz Mini Games é uma plataforma de mini jogos de navegador criada no Brasil. O objetivo é oferecer diversão rápida e gratuita para quem quer testar conhecimentos de música, vocabulário e muito mais.

Todos os jogos funcionam 100% no navegador — sem instalação, sem cadastro obrigatório. Basta acessar e jogar.

🕹️ Os jogos

🎵

Guess The Song

Ouça o trecho e adivinhe a música antes do tempo acabar. Escolha o gênero ou busque um artista. Ranking global e quizzes personalizados.

🟩

Palavras BR

Adivinhe a palavra do dia em até 6 tentativas. Quatro modos progressivos: Solo, Duo, Quarteto e Desafio. Ranking diário.

⚙️ Como é feito

O projeto é desenvolvido com tecnologias web modernas:

  • Backend: Java com Spring Boot, banco de dados SQLite.
  • Frontend: HTML, CSS e JavaScript puro — sem frameworks, sem build tools.
  • Música: Prévias de áudio fornecidas pela API do Deezer.
  • Hospedagem: Servidor próprio, infraestrutura em nuvem.

O código é mantido de forma independente, sem financiamento externo. Os anúncios exibidos pelo Google AdSense ajudam a cobrir os custos de servidor.

Não coletamos dados sensíveis. O único dado que armazenamos é o nome de jogador (digitado por você) e sua pontuação, usados exclusivamente para o ranking público.
